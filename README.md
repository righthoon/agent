# 약속 (YakSok) — 복약안전 AI 앱

> **약을 안심하고 드실 수 있게.** 약봉투 사진 한 장으로 병용금기와 주의 음식을 확인하고, AI 챗봇에게 물어보는 복약안전 도우미입니다.

고령·다약제 복용이 늘면서 여러 병원에서 받은 약이 서로 부딪히는 **병용금기**가 큰 문제입니다. 약속은 공식 데이터(한국의약품안전관리원 DUR)를 근거로, 위험 여부를 **AI가 지어내지 않고** 규칙엔진으로 판정합니다.

---

## 주요 기능

| 기능 | 설명 |
|------|------|
| 📷 **약봉투/처방전 사진 인식 (OCR)** | Claude 비전으로 사진에서 약 이름을 자동 추출해 복용약 목록에 채웁니다. |
| ⚠️ **병용금기 판정** | 복용약들을 공식 DUR 병용금기 데이터와 대조해, 위험하면 이유와 함께 경고합니다. (결정론적 규칙엔진 = 안전 코어) |
| 💬 **AI 챗봇 상담** | "내 약들 같이 먹어도 돼?" 같은 질문에 도구(tool)를 호출해 근거 기반으로 답하고, 약-음식 상호작용과 일반 건강 질문도 안내합니다. |

> 모든 답변은 **의료행위가 아닌 정보 제공**이며, 마지막에 약사·의사 상담을 권합니다.

---

## 기술 스택

- **언어**: Python 3.13
- **화면**: Streamlit
- **AI**: Anthropic Claude (`claude-sonnet-5`) — 비전(OCR) + 도구 호출(tool use)
- **약 이름 매칭**: rapidfuzz
- **데이터**: 한국의약품안전관리원 병용금기(DUR)

핵심 설계: **위험 판정은 LLM이 아니라 결정론적 코드가 담당**합니다. Claude는 사진을 약 이름으로 바꾸고, 확정된 결과를 쉽게 설명하는 역할만 합니다.

## 실행 방법

### 1) 패키지 설치
```bash
pip install streamlit anthropic python-dotenv rapidfuzz
```

### 2) API 키 설정
```bash
# 예시 파일을 복사해서 .env 를 만듭니다
cp .env.example .env        # Windows: copy .env.example .env
```
그런 다음 `.env` 파일을 열어 본인의 Anthropic API 키를 넣습니다:
```
ANTHROPIC_API_KEY=sk-ant-...(본인 키)...
```
> API 키는 https://console.anthropic.com 에서 발급받습니다. `.env` 는 `.gitignore` 로 보호되어 공개되지 않습니다.

### 3) 공공데이터 CSV 준비
아래 [공공데이터 받기](#공공데이터-받기) 를 참고해 CSV를 내려받아 **프로젝트 루트**(`app.py` 와 같은 폴더)에 둡니다.

### 4) 실행
```bash
streamlit run app.py
```
브라우저에서 `http://localhost:8501` 이 자동으로 열립니다.

## 공공데이터 받기

앱 실행에는 병용금기 데이터 파일이 필요합니다. (용량이 커서 저장소에는 포함하지 않습니다.)

- **데이터 이름**: 한국의약품안전관리원_병용금기약물
- **받는 곳**: [공공데이터포털(data.go.kr)](https://www.data.go.kr) 에서 **"병용금기"** 또는 **"한국의약품안전관리원 병용금기"** 검색
- **파일명**: `한국의약품안전관리원_병용금기약물_20240625.csv`
- **놓을 위치**: 프로젝트 루트 (예: `약속/한국의약품안전관리원_병용금기약물_20240625.csv`)
- **인코딩**: `cp949` — 코드가 자동으로 판별해 읽습니다.

> 파일명이 다르면 `chatbot_core.py`, `pharma_chatbot.py` 상단의 `CSV = "..."` 값을 실제 파일명에 맞춰 주세요.
